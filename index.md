# My Prompt Library

A personal library of reusable prompts for Claude and Claude Code.

## How to Use
- **Claude.ai**: Paste this at the start of a conversation:
  "Fetch my prompt library index from: https://raw.githubusercontent.com/dpersaud91/MyPromptLibrary/refs/heads/main/index.md"
- **Claude Code**: Automatically loaded via ~/.claude/CLAUDE.md

## Categories

### 💻 Coding — `coding.md`
Code review, write tests, explain code, refactor, debug, write a script, Claude Code tasks

### ✏️ Editing — `editing.md`
Proofread, improve clarity, change tone, tighten writing, structural edit, line edit

### 🧠 Thinking — `thinking.md`
Brainstorm, decision making, challenge my thinking, first principles, think out loud, blind spots, pros and cons

### 🔍 Research — `research.md`
Explain a topic, research and summarize, compare options, summarize notes, find weaknesses in an argument, literature overview

### ✍️ Writing — `writing.md`
Write a blog post, write from an outline, write a draft from notes, write an introduction, write a conclusion, draft an email, continue my writing
